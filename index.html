Machine Learning

How the Model Works

The recommender uses a hybrid KNN model combining three complementary signals — genre labels, semantic meaning, and structural metadata — into a single weighted feature matrix.

Each show is represented as a vector. The model runs cosine similarity across ~8,800 Netflix titles using scikit-learn's NearestNeighbors to find the 50 closest matches, then paginates them in sets of 10.

For titles not in the 2021 dataset, a TMDB fallback fetches live genre and description data, builds a feature vector using the same fitted transformers, and queries it against the local KNN index in real time.

TF-IDF · Genre Labels 30%
Bigram TF-IDF on the listed_in column · 50 features
Sentence-BERT · Description 50%
all-MiniLM-L6-v2 · 384-dim embeddings · L2 normalized
Metadata · OHE + Scaled 20%
Type, rating (OHE) + release year, duration (StandardScaler)

Evaluation

Model Performance

TF-IDF Baseline · Genre Accuracy
89.3%
High label accuracy — but essentially just a genre filter. Diversity distance of only 0.0046.
Hybrid Model · Genre Accuracy
67.7%
Lower label accuracy — intentionally. The model recommends across genre boundaries when narratives are similar.
Hybrid Model · Diversity Distance
0.44
vs 0.0046 for TF-IDF baseline. 95× more diverse — finds semantically meaningful recommendations.
Dataset Size
8,800
Netflix shows and movies from Kaggle CSV. Data covers content up to 2021.
Key insight: the 22% drop in genre accuracy is a feature, not a bug. A pure genre filter would just return the same category. The hybrid model crosses genre lines when the narrative, tone, or themes are similar — which is how humans actually recommend shows.

Request Flow

🌐
User Browser
Types a show title → autocomplete fetches from /titles endpoint → selects and submits
▲
Vercel — Next.js Frontend
Renders the UI, calls the FastAPI backend, handles pagination state and poster progressive loading
🤗
HuggingFace Spaces — FastAPI Backend
Docker-deployed Python service. Loads KNN model on startup. Routes to local DB or TMDB depending on whether the title exists in the 2021 dataset
🎬
TMDB API (fallback)
For post-2021 titles: fetches genres + description, builds live feature vector using same fitted transformers, queries local KNN index
🧠
KNN Model — Local Index
Cosine similarity over 8,800 pre-computed hybrid vectors. Returns 50 neighbors, paginated in sets of 10 per request
